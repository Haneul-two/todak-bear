Why Todak

다그치는 펫이 아니라, 곁에 있는 곰

흔한 코딩 펫은 “잘못하면 화내는” 캐릭터예요. 토닥곰은 정반대 방향을 골랐습니다.

😤 보통의 펫

  • 에러 나면 토라지고 체력이 깎임
  • 오래 멈추면 삐지거나 죽음
  • “또 틀렸네” 식의 다그침
  • 실수를 벌점으로 환산

토닥곰 데려오기

바로 받아 실행하거나, 소스에서 직접 빌드할 수 있어요.

⬇ 다운로드 (Windows)
🛠 소스에서 빌드
👀 브라우저로 먼저 보기
  1. 설치 파일 받기 아래 버튼으로 최신 Windows 설치 파일(todak-bear_x64-setup.exe, 약 8 MB)을 내려받으세요.
    ⬇ Windows 설치 파일 모든 릴리즈 보기
  2. 실행 내려받은 설치 파일을 실행합니다. Windows SmartScreen 경고가 뜨면 추가 정보 → 실행을 눌러주세요(서명 전 단계라 정상입니다).
  3. 곰 만나기 바탕화면 오른쪽 아래에 토닥곰이 나타나요. 좌클릭=표정 구경, 우클릭=메뉴(산책 끄기·위치 초기화·종료), 잡고 끌어 옮길 수 있어요.
ⓘ 아직 공개 설치 파일이 게시되지 않았다면 Releases가 비어 있을 수 있어요. 그럴 땐 아래 소스에서 빌드 탭으로 바로 실행할 수 있습니다.
  1. 준비물 설치 Node.js와 Rust(rustup)를 설치합니다. Windows는 빌드 도구(MSVC)도 필요해요.
  2. 레포 클론 & 의존성 
    # 저장소 받기
git clone https://github.com/Haneul-two/todak-bear
cd todak-bear/apps/desktop
npm install
  3. 빌드 & 실행 
    # 곰 에셋을 dist로 조립한 뒤 데스크톱 앱 실행
node build-dist.js
npm run dev
ⓘ 데스크톱 앱(apps/desktop)이 메인 표면이에요. 터미널 곰·GitHub 카드도 같은 레포에서 node 한 줄로 바로 체험할 수 있습니다.
  1. 설치 없이 코어 데모 Rust 없이도 곰의 모든 기분을 브라우저/터미널로 미리 볼 수 있어요. 
    # 모든 기분 한 번에
node packages/bear-core/demo.js

# 데스크톱 UI를 브라우저로 (Rust 불필요)
node apps/desktop/build-dist.js
# → apps/desktop/dist/index.html 을 브라우저로 열기
  2. 마음에 들면 위 다운로드 또는 소스에서 빌드 탭으로 진짜 바탕화면 펫을 데려오세요. 🐾
